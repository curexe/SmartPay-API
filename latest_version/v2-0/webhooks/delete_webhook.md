
# Delete Webhook

~~~
DELETE https://api.smartpay.curexe.com/2-0/webhooks/{{WEBHOOK_ID}}
~~~
* replace {{WEBHOOK_ID}} with the actual webhook_id

## Description

Deletes a webhook.
